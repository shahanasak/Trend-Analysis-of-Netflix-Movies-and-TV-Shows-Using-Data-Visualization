# Trend-Analysis-of-Netflix-Movies-and-TV-Shows-Using-Data-Visualization.
This project analyzes Netflix movies and TV shows using data visualization. Data was cleaned in Excel and visualized in Microsoft Power BI using charts and slicers to show trends, genres, and ratings, helping generate meaningful insights for decision-making.
# CONTENT OF PROJECT:This project focuses on analyzing trends in Netflix movies and TV shows using data visualization techniques.
With the rapid growth of digital streaming platforms, understanding content trends has become essential for identifying audience preferences and industry patterns.
The dataset used in this project contains detailed information about Netflix titles, including type, genre, release year, runtime, and audience ratings.
The first stage of the project involved data cleaning and preprocessing using Microsoft Excel. Several important steps were performed to ensure the quality and consistency of the data.
These included removing duplicate records, handling missing values in columns such as IMDb votes and seasons, correcting encoding errors in text fields, and standardizing formats for numerical and categorical data. Additional transformations, such as creating calculated fields and organizing the dataset, were also carried out to make the data suitable for analysis.
After preprocessing, the cleaned dataset was imported into Microsoft Power BI to build an interactive and visually appealing dashboard
Various types of visualizations were used, including bar charts to compare movies and TV shows, line charts to analyze content growth over time, pie charts to represent genre distribution, and card visuals to display key metrics such as total titles and average ratings.
Interactive features such as slicers and filters were added to enhance user experience, allowing dynamic exploration of the data. Overall, this project demonstrates how data visualization tools can effectively transform raw data into meaningful insights, supporting better analysis and decision-making.
# RESULT:The analysis of Netflix movies and TV shows using data visualization provided several meaningful insights into content trends and audience preferences.
The results show that Netflix content has grown significantly over the years, with a noticeable increase in the number of titles released after 2015.
Movies were found to be more dominant compared to TV shows on the platform. Genre analysis revealed that Drama, Comedy, and Action are among the most popular categories, indicating strong audience demand for these types of content.
The relationship between IMDb ratings and votes suggests that highly rated content generally receives more audience engagement. Additionally, the analysis highlighted variations in content production across different years and categories, helping to identify peak periods of growth.
The interactive dashboard created using Microsoft Power BI allowed dynamic filtering and exploration of the data, making it easier to uncover patterns and trends. Users can analyze content based on year, genre, and type, enhancing the overall usability of the report.
Overall, the project successfully demonstrates how data cleaning in Excel and visualization in Power BI can be used to transform raw data into actionable insights. The findings can help in understanding Netflix’s content strategy and audience preferences, supporting better decision-making and analysis.
